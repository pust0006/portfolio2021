What you will do:

  1. Follow a Flask tutorial to acces a CSV file.  
  2. Create a Flask web application that interacts with a database.

Task 1:

  • Download and install the Chrome Github markdown extension reader: https://chrome.google.com/webstore/detail/markdown-reader/gpoigdifkoadgajcincpilkjmejcaanc
  • Open Chrome, click on the Markdown Reader icon on the top right of your Chrome window, then select Manage Extensions. Locate the "allow access to File URL" and enable it.
  • Do the Lab 11 exercise available on brightspace. The file is a github md file so the markdown reader must be downloaded and installed. The zip file you need is available on Brightspace.    

Task 2:

  • This task uses the week10.db sqlite3 database from last lab. You also need to install the wt_forms module using pip3. More information will be given in class and in lab.
  • Using Pycharm,  create a flask application with three routes:
    • Default page displays a simple menu:
      Option 1 of the menu is used to display the Google map of the city of a specific student.
      Option 2 of the menu displays a table of all students, country and cities.
    • the findStudent page displays a form where the user can select a student and the page then opens a Google map of the city from where the student is from.         
    • The displayAll page displays a simple table of all students, cities and countries stored in the database. Each student name is also a hyperlink that opens in a new window a Google map of the city from where the student is from. Bonus: The table headings sort the table according to the heading.

Task 3: Submit your lab

  • Upload the Zipped Pycharm project from Task2 to Github and make me a collaborator. Upload the Github link to Brightspace.
